Proyecto Tasks

El proyecto de Tareas es una API diseñada para permitir la administración de tareas, con operaciones CRUD (Crear, Leer, Actualizar, Eliminar) sobre una base de datos en Firestore. El proyecto utiliza tecnologías como Firebase y un modelo de tareas en JavaScript que aprovecha el sistema de colección de Firestore para el almacenamiento y recuperación de datos. Las tareas son identificadas de forma única con un UUID, lo cual facilita su manipulación y trazabilidad.

Funcionalidades de la API

Esta API incluye cinco endpoints principales:

  • Crear Tarea (POST): Permite agregar una nueva tarea a la base de datos.
  • Listar Tareas (GET): Devuelve todas las tareas registradas en la base de datos.
  • Obtener Tarea por ID (GET): Recupera la información de una tarea específica mediante su ID.
  • Editar Tarea (PUT): Actualiza la información de una tarea existente.
  • Eliminar Tarea (DELETE): Elimina una tarea de la base de datos de forma permanente.

A lo largo de esta documentación, se proporciona una descripción detallada de cada endpoint, los parámetros de entrada y salida, así como ejemplos de uso.
