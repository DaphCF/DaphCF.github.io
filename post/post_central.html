Crear Tarea

Permite agregar una nueva tarea con título, descripción y estado de finalización.

Authorizations:

No requiere autorización.

Request Body schema: required

  • tittle (required): string

    • Titulo de la tarea.

  • description (required): string

    • Descripción de la tarea.

  • completed (required): boolean

    • Estado de finalización de la tarea.

Responses

201

Un objeto Task creado.

RESPONSE SCHEMA: application/json

  • id (required): string

    • Identificador único de la tarea creada.

  • tittle (required): string

    • Titulo de la tarea.

  • description (required): string

    • Descripción de la tarea.

  • completed (required): boolean

    • Estado de finalización de la tarea.

  • createdAt (required): string (ISO 8601 date)

    • Fecha y hora de creación de la tarea en formato ISO.
