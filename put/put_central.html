Editar Tarea

Actualiza la información de una tarea existente, permitiendo modificar título, descripción y estado.

Authorizations:

No requiere autorización.

Path parameters:

  • id (en la URL): ID de la tarea.

Request Body schema: required

  • tittle (required): string

    • Titulo de la tarea.

  • description (required): string

    • Descripción de la tarea.

  • completed (required): boolean

    • Estado de finalización de la tarea.

Responses

200

Objeto Task editado correctamente

RESPONSE SCHEMA: application/json

  • code (required): integer

    • Código de respuesta HTTP (200 si se actualiza correctamente).

  • message (required): string

    • Mensaje de confirmación..

  • task (required): object

    Datos de la tarea actualizada.

    • id (required): string

      • Identificador único de la tarea.

    • tittle (required): string

      • Título actualizado de la tarea.

    • description (required): string

      • Descripción actualizada de la tarea.

    • completed (required): boolean

      • Estado de finalización de la tarea.

404

Error en la petición

RESPONSE SCHEMA: application/json

  • code (required): integer

    • Código de error HTTP (404 en caso de error).

  • message (required): string

    • Mensaje descriptivo del error.
