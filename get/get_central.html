Listar Tareas

Devuelve una lista de todas las tareas almacenadas en la base de datos.

Authorizations:

No requiere autorización.

Request Body schema:

No requiere una entrada.

Responses

200

Lista de objetos Task en el sistema.

RESPONSE SCHEMA: application/json []: array

  • id (required): string

    • Identificador único de la tarea.

  • tittle (required): string

    • Titulo de la tarea.

  • description (required): string

    • Descripción de la tarea.

  • completed (required): boolean

    • Estado de finalización de la tarea.

  • createdAt (required): string (ISO 8601 date)

    • Fecha y hora de creación de la tarea en formato ISO.

404

Error en parámetros de la petición

RESPONSE SCHEMA: application/json

  • code (required): integer

    • Código de error HTTP (404 en caso de error).

  • message (required): string

    • Mensaje descriptivo del error.
