Eliminar Tarea

Elimina una tarea específica de la base de datos mediante su ID de forma permanente.

Authorizations:

No requiere autorización.

Path parameters:

  • id (en la URL): ID de la tarea a eliminar.

Responses

200

Objeto Task eliminado correctamente

RESPONSE SCHEMA: application/json

  • code (required): integer

    • Código de respuesta HTTP (200 si se elimina correctamente).

  • message (required): string

    • Mensaje de confirmación.

404

Error en la petición

RESPONSE SCHEMA: application/json

  • code (required): integer

    • Código de error HTTP (404 en caso de error).

  • message (required): string

    • Mensaje descriptivo del error.
