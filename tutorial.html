Tutorial de Uso - Proyecto Tareas

Aprende cómo manejar las tareas en esta aplicación web

Enlace a GitHub: Proyecto_Task

Enlace a GitHub: Interfaz_Tasks

1. Iniciar sesión

Para iniciar sesión en la aplicación web:

  • Introduce tu usuario y contraseña en los campos correspondientes.
  • Haz clic en el botón "Iniciar sesión".
  • Si el usuario y contraseña son correctos podrás acceder sl gestor de tareas.

    • Usuarios = ['admin', 'usuario1', 'usuario2']
    Contraseñas = ['admin', 'contra1', 'contra2']

2. Crear una Tarea

Para crear una nueva tarea:

  • Haz clic en el botón "Agregar una tarea".
  • Introduce el título y la descripción de la tarea.
  • Presiona la tecla enter para crear la tarea.

3. Ver Lista de Tareas

Para ver todas las tareas creadas:

  • Podrás ver todas las tareas con su título, descripción y estado desde la página principal.

4. Ver una Tarea por ID

Para ver una tarea específica por su ID:

  • En la lista de tareas, introduce el ID en el campo correspondiente.
  • Presiona la tecla enter para ver los detalles de la tarea.

5. Marcar una Tarea como Completada

Para marcar una tarea como completada:

  • Selecciona la tarea de la lista.
  • Haz clic en el botón ✔.
  • La tarea se cambiara a formato tachado para indicar que ha sido completada.

6. Editar una Tarea

Para editar el título de una tarea:

  • Selecciona la tarea que deseas editar en la lista de tareas.
  • Haz clic en el botón ✎.
  • Modifica el título o la descripción en los campos correspondientes.
  • Presiona la tecla enter para actualizar la tarea.

7. Eliminar una Tarea

Para eliminar una tarea:

  • Selecciona la tarea que deseas eliminar de la lista.
  • Haz clic en el botón ✘.
  • Se eliminará la tarea de forma permanente.

8. Cerrar sesión

Para cerrar sesión en la aplicación web:

  • Haz clic en el botón "Cerrar sesión".
  • Regresarás al inicio de sesión.
